格式化并挂载数据盘

下面使用 fdist 查看数据盘并分区。简单说明一下使用方法与步骤。
查看所有数据盘：fdisk -l
对数据盘分区：fdisk -S 56 /dev/xvdb，再依次输入 n p 1 enter enter wq
格式化分区：mkfs.ext3 /dev/xvdb1
创建挂载目录：mkdir /data
添加分区信息：echo '/dev/xvdb1 /lgdata ext3 defaults 0 0' >> /etc/fstab
挂载：mount -a
